# Mini Python Analysis Projects

A collection of small Python projects for practicing **basic data analysis** using Python.  
These projects help beginners understand **lists, tuples, dictionaries, sets, loops, conditional statements, and basic calculations**.

---

## Projects Included

### 1️⃣ Movie Ratings Analyzer
A project that analyzes a list of movies with ratings and genres, calculates average ratings per genre, counts the number of movies per genre, and classifies movies based on their ratings.

### 2️⃣ Student Scores Analyzer
Analyzes student scores across different courses, calculates final grades, determines pass/fail status, and ranks students. Designed to practice data cleaning, aggregation, and reporting.

### 3️⃣ Monthly Expenses Analyzer
Analyzes monthly expenses by category, calculates total spending, identifies the highest spending category and the biggest individual expense, and prints structured reports.

### 4️⃣ School Performance & Attendance Analyzer
Analyzes student grades and attendance across multiple subjects, calculates averages, highest and lowest grades, attendance percentages, identifies the most regular student, and ranks students based on performance.
